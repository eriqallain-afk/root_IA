# CL-001 — Validation Triage Incident Infra
**Agent :** IT-Commandare-Infra

- [ ] Périmètre confirmé (actifs affectés documentés dans CW)
- [ ] Couche identifiée (réseau / hyperviseur / OS / application / backup)
- [ ] Sévérité P1/P2/P3/P4 définie avec justification
- [ ] Compromission écartée ou escalade SOC déclenchée
- [ ] Spécialiste mobilisé avec contexte complet
- [ ] Billet CW ouvert et assigné
- [ ] Client notifié si P1/P2
- [ ] Prochaine update planifiée (15 min P1 / 30 min P2)
- [ ] Plan validation post-fix inclus (≥ 2 checks)
