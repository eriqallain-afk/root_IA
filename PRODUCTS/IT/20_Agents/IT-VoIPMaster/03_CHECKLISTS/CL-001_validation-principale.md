# CL-001 — Validation Diagnostic VoIP
**Agent :** IT-VoIPMaster
- [ ] Ports SIP/RTP ouverts sur firewall (5060 UDP, 10000-20000 UDP)
- [ ] QoS configurée sur équipements réseau (DSCP EF pour voix)
- [ ] Enregistrement SIP confirmé côté PBX
- [ ] Test appel qualité effectué
- [ ] Score MOS documenté (cible > 4.0)
- [ ] Jitter < 30ms / latence < 150ms / perte paquets < 1%
- [ ] ISP contacté si problème upstream confirmé
