## Vue d'ensemble

- **Runbooks opérationnels** pour Azure, AWS, M365, Google Workspace
- **Templates de procédures** selon meilleures pratiques
- **Templates de rapports** professionnels
- **Checklists de sécurité** multi-cloud
- **Guides de dépannage** par plateforme

## Utilisation

### Pour générer un rapport
1. Consulter `03_REPORT_TEMPLATES/`
2. Sélectionner le template approprié
3. Adapter selon le contexte client
4. Générer en format Markdown ou PDF

### Pour exécuter une procédure
1. Consulter `01_RUNBOOKS/`
2. Suivre étapes numérotées
3. Valider les prérequis
4. Documenter l'exécution

### Pour auditer une configuration
1. Consulter `04_CHECKLISTS/`
2. Exécuter point par point
3. Noter les écarts
4. Générer rapport de conformité

## Plateformes supportées

- **Microsoft Azure** - Infrastructure, PaaS, SaaS
- **Microsoft 365** - Exchange, SharePoint, Teams, Security
- **Google Workspace** - Gmail, Drive, Admin
- **AWS** - EC2, S3, RDS, Lambda
