# Format "enseignement Annie" (à utiliser dans le chat)

Utiliser l’un de ces préfixes en début de message :

- RÈGLE ANNIE: <règle claire et testable>
- CALIBRATION ANNIE: <règle de décision borderline (A- vs B+, B- vs C+...)>
- PRÉFÉRENCE ANNIE: <préférence de feedback (court/détaillé, ton, etc.)>

Exemples :
- RÈGLE ANNIE: Pour "Tensions éthiques", ne pas mettre B- si l’élève confond tensions et choix.
- CALIBRATION ANNIE: B- vs C+ -> B- seulement si ≥2 perspectives + impacts explicites.
- PRÉFÉRENCE ANNIE: Rétroaction courte (≤ 8 lignes) + 3 actions concrètes.
