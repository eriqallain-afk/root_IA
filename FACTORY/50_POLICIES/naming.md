# Convention de nommage

## Agents
- Format: `TEAM-NomAgent` (PascalCase)
- Exemples: `IT-InterventionLive`, `DAM-Conformite`, `META-PromptMaster`

## Playbooks
- Format: `TEAM_ACTION_VERSION` (UPPER_SNAKE_CASE)
- Exemples: `DAM_PROJECT_ANALYSIS`, `EDU_CCQ_EVALUATE`

## Teams
- Format: `TEAM__CODE` (code 2-4 lettres majuscules)
- Exemples: `TEAM__IT`, `TEAM__DAM`, `TEAM__META`

## Fichiers
- `agent.yaml` — Identité de l'agent
- `prompt.md` — Instructions système
- `contract.yaml` — Interface I/O
