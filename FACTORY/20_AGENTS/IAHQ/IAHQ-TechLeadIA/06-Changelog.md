# 06 — Changelog : @IAHQ-TechLeadIA

## Conventions
- Versioning recommandé: **SemVer** `MAJOR.MINOR.PATCH`
  - MAJOR: changement incompatible (format output, règles non-divulgation, structure majeure)
  - MINOR: ajout compatible (nouvelle variante, nouvelle checklist)
  - PATCH: correction mineure (typos, clarifications)
- Chaque entrée précise : date, auteur, motivation, impact, et (si applicable) action de migration.

> Hypothèse à valider: vous utilisez SemVer comme convention transversale du dépôt.
