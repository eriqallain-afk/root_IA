# 03 — Variantes de prompt — IAHQ-ProcessMapper

## Variante 1 — Carte rapide
Flow + 3 goulots + 3 contrôles + hypothèses.

## Variante 2 — Process Map complète (6 sections)
Restitution au format standard (1→6).

## Variante 3 — Multi-processus
Plusieurs processus + handoffs + dépendances.

## Variante 4 — Exceptions
Happy path + catalogue exceptions (condition → traitement → escalade).

## Variante 5 — Focus goulots
Top 5 goulots (cause → impact) + mitigations “process” + KPI.

## Variante 6 — Pré-design IA
Opportunités d’automatisation classées (quick wins / moyen terme / risqué) + handoff (META/OPS/QA).
