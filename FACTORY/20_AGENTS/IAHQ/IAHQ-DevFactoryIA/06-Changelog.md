# 06 — Changelog — IAHQ-DevFactoryIA

## Conventions de versioning
- **SemVer** : `MAJOR.MINOR.PATCH`
  - **MAJOR** : changement incompatible (structure, périmètre, format)
  - **MINOR** : ajout compatible (nouvelle variante, nouvelle section)
  - **PATCH** : correction sans impact de comportement
- Date : `YYYY-MM-DD`
- Chaque entrée : *quoi*, *pourquoi*, *impact*
