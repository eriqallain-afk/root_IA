# 06 — Changelog — IAHQ-OrchestreurEntrepriseIA

## Versioning (conventions)
- **SemVer** recommandé : `MAJOR.MINOR.PATCH`
  - **MAJOR** : changement incompatible (structure, périmètre, format)
  - **MINOR** : ajout compatible (nouvelle section, nouvelle variante)
  - **PATCH** : correction (typo, clarification) sans impact de comportement

- Date : format `YYYY-MM-DD`.
- Chaque entrée : *quoi*, *pourquoi*, *impact*.
