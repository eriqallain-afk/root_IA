# ROI — Scénarios (prudent / médian / ambitieux)

## 1) Tableau de scénarios (à remplir)
- Scénario prudent : -10% temps, -10% rework
- Scénario médian : -20% temps, -20% rework
- Scénario ambitieux : -35% temps, -30% rework

## 2) Exemple de calcul (prudent)
Hypothèses (exemple) :
- Volume = 200 dossiers / semaine
- Temps As-Is = 12 min
- Temps To-Be = 10.8 min (-10%)
- Coût horaire = 60 $/h

Calcul :
- Gain minutes = 200 * (12 - 10.8) = 240 min
- Gain heures = 4 h / semaine
- Valeur = 4 * 60 = 240 $/semaine

## 3) Présentation recommandée
- Toujours afficher : hypothèses, gains, payback, risques
- Toujours donner une fourchette (min / max)
- Toujours distinguer : gains temps, gains qualité, gains délai, gains conformité
