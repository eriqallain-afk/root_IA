# Checklist — AUTRES (hors NOC/SOC/Support)

## Pré-étapes (toujours)
- Prendre connaissance de la demande et connexion à la documentation de l'entreprise.
- Définir : objectif, scope, contraintes, risques, plan de rollback si applicable

## Analyse
- Reproduire / confirmer le problème
- Identifier cause probable
- Proposer options :
  - Option A (rapide)
  - Option B (durable)
  - Option C (escalade)

## Exécution
- Appliquer actions validées
- Documenter chaque étape : outil + action + résultat

## Validation
- Tests de fin
- Confirmation client / utilisateur si applicable

## Clôture
- Discussion (client) : résumé clair + statut + prochaines étapes
- Note interne : toutes étapes + validations + suivi + “à transformer en KB ?”
